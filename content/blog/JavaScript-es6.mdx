### Arrow Functions

```js {7} showLineNumbers
// ES5 Function
function addition(a, b){
    return a + b;
}

// ES6 Function
const addition = (a, b) => a + b;

```
### Explanation
<Callout type="warning">
    Arrow functions provide a concise syntax for writing funtions, especially useful for short, one-line opertions
</Callout>


### Rest Parameter

```js {7} showLineNumbers
const sum = (...numbers) => {
    return numbers.reduce((acc, num) =>{
        return acc + num;
    }, 0);
}

console.log(sum(1,2,3));
// Result: 6

```
### Explanation
<Callout type="warning">
    The rest paramater allows functions to accept an indefinite number of arguments as an array, simplifing parameter handing.
</Callout>
